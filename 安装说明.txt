## 文本攻击修复系统 - 安装说明

### 系统要求
- Python 3.8 或更高版本
- 稳定的网络连接

### 安装步骤

1. **安装依赖**
   - 双击 `install.bat` 文件
   - 等待依赖安装完成
   - 如果安装失败，可以尝试以管理员身份运行

2. **启动系统**
   - 双击 `start.bat` 文件
   - 系统会自动在默认浏览器中打开应用程序
   - 如果浏览器没有自动打开，可以手动访问: http://localhost:8501

### 可能遇到的问题

1. **无法安装依赖**
   - 确保网络连接正常
   - 尝试手动安装：
     ```
     pip install -r requirements.txt
     ```
   - 如果出现权限问题，尝试：
     ```
     pip install -r requirements.txt --user
     ```

2. **应用程序启动失败**
   - 确保所有依赖都已正确安装
   - 检查Python版本是否满足要求
   - 尝试手动启动：
     ```
     streamlit run app.py
     ```

3. **界面加载缓慢**
   - 这是正常现象，第一次启动时需要加载资源
   - 后续启动将会更快

### 数据集说明

本系统使用Enron数据集作为示例：
- 原始数据位于 `dataset/enron/` 目录
- 攻击后的数据将保存在 `dataset/enron_attack/` 目录
- 修复结果将保存在 `dataset/repair_results/` 目录

如需使用自己的数据集，请确保保持相同的目录结构。
